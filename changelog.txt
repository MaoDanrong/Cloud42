Cloud42 v1.2.0  (2010-01-14)
============================

- fixed a bug that caused loading of wrong configuration in some cases

- New feature: added the ability so switch between the different AWS Regions
	- both in wep application and in web service
	- new method "listRegions" in Web service lists all available Regions, URL of target region must be specified in each call


Cloud42 v1.1.0  (2009-11-10)

- updated Typica and added support for Eucalyptus
	- endpoint now can be configured in the file core/module-configuration/src/main/resources/config.properties
